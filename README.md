# FitnessAI 💪🤖

An AI-powered mobile fitness coach built with React Native and Expo.

## Vision
FitnessAI is your intelligent training partner — tracking calories, 
steps, and goals while delivering personalized coaching powered by 
the Anthropic AI API.

## Features (In Development)
- 🏋️ AI-powered workout planning
- 🥗 Natural language food logging
- 📊 Calorie and step tracking
- 🎯 Personalized goal setting
- 💬 Adaptive AI coaching

## Tech Stack
- React Native
- Expo
- TypeScript
- Anthropic API
- Python (Backend)

## Getting Started
```bash
npm install
npx expo start
```
